# Jogo de Blackjack!

## Sobre o código

O código é feito todo em Java e há também um arquivo cards.txt que faz o "desenho" das cartas do baralho.
O objetivo do código é treinar lógica de programação e a sintaxe do Java. Utiliza-se if-else, while loop, interação com o usuário (Scanner) e mais.
E o jogo é bem divertido de jogar!

## Regras:

No Blackjack, é o jogador contra o dealer.

O dealer dá ao jogador duas cartas viradas para cima.

O dealer também começa com duas cartas, mas apenas uma delas fica virada para cima, pois a outra fica virada para baixo.

Primeiro, é a vez do jogador. O jogador pode decidir entre "hit" ou "stay".

Se escolher "hit": o jogador compra outra carta.

Se escolher "stay": o jogador não compra nenhuma carta.

Se o jogador decidir por "hit" e o valor da sua mão somada ultrapassar 21, perde.

E quando o jogador escolher "stay", o dealer revela a sua própria carta escondida.

Em seguida, o dealer deve pedir cartas até que o total da sua mão chegue a 17. Com 17 pontos ou mais, o dealer deve parar.

Você ganha se o valor da sua mão for maior que o da mão do dealer ou se por acaso a mão do dealer estourar (ultrapassar 21).

Mas se a mão do dealer foi mais que a sua, você perde.
